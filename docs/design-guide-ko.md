# 디자인 가이드 (한국어)

- 보조 텍스트: `#334155`
- 강조(Primary): `#2563EB`
- 성공(완료): `#16A34A`
- 경고: `#D97706`

## 2) 타이포그래피
- 제목: Pretendard/Noto Sans KR 700
- 본문: Pretendard/Noto Sans KR 400~500
- 숫자(시간): tabular nums 활성화

## 3) 핵심 컴포넌트
- **시간순 타임라인 카드**: 시작시간, 종료시간, 카테고리 칩, 체크박스
- **체크리스트**: 우선순위/반복 아이콘 표시
- **원형 스케줄 뷰**: 시간 블록 + 완료율 컬러 채움
- **업무/사적 세그먼트**: 상단 고정 토글
- **카테고리 에디터**: 직업별 템플릿 + 사용자 커스텀

## 4) 인터랙션
- 완료 체크 시 미세 애니메이션(120~180ms)
- 알림 ON/OFF 즉시 토스트 피드백
- 뷰 전환(데일리/위클리/먼슬리/이얼리)은 상태 유지

## 5) 대시보드 UX
- 이번 주 완료율(큰 숫자 + 도넛)
- 카테고리별 시간 누적 막대
- 반복 일정 성공률 히트맵

## 6) 온보딩
- 사용자 유형 선택: 프리랜서 / 학생 / 직장인
- 추천 카테고리 자동 생성
- 알림 권한 요청은 첫 일정 생성 직후
